# Validation Checklist After Upgrade
- [ ] Trial Balance before upgrade == Trial Balance after upgrade
- [ ] No journal hashes changed for existing records
- [ ] No orphan records created
- [ ] VAT reports for old periods unchanged
- [ ] Public API endpoints behave as before
- [ ] Audit trail intact with hash chain continuity
- [ ] Migration scripts versioned and rollback scripts ready
- [ ] Monitoring active and alerts configured
- [ ] Backups verified and restorable
