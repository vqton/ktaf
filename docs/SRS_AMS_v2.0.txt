8.2 Use Case Specification - UC-DB-003 (Brief - PostgreSQL Auth)
MụcChi tiếtUC-IDUC-DB-003TênCấu hình PostgreSQL Authentication (SSPI hoặc Username+Password)Actor chínhDBAdminActor phụITAdmin (cấp thông tin AD domain)Mức độBrief - xem Use Case Register Excel cho Fully DressedPre-conditionPostgreSQL 16 đã cài, service đang chạy, pg_hba.conf và pg_ident.conf có thể sửaTriggerAdmin đã chọn DatabaseProvider=PostgreSql trong appsettings.jsonMain Flow - Method A (SSPI)1. DBAdmin sửa pg_ident.conf: map DOMAIN\ams_app_user → ams_pg_user 2. Sửa pg_hba.conf: thêm dòng host all ams_pg_user 127.0.0.1/32 sspi include_realm=0 3. Restart PostgreSQL service 4. Test: psql -U ams_pg_user -h localhost (tự động login không cần password) 5. Cập nhật ConnectionStrings trong appsettings.json: thêm Integrated Security=trueMain Flow - Method B (pg login)1. DBAdmin tạo role: CREATE ROLE ams_user LOGIN PASSWORD 'strong_pwd'; 2. Sửa pg_hba.conf: host all ams_user 127.0.0.1/32 scram-sha-256 3. REVOKE login từ PUBLIC trên database ams_db 4. GRANT CONNECT, USAGE trên các schemas cho ams_user 5. Cập nhật ConnectionStrings trong appsettings.json: Username=ams_user;Password=...Alternative FlowMethod A lỗi (AD không cấu hình Kerberos) → tự động fallback sang Method BExceptionpg_hba.conf syntax error → PostgreSQL service không start → xem logs pg_log/Post-conditionApplication kết nối DB thành công. Health Check /api/health/db trả về Healthy.FR liên quanFR-DB-003, NFR-DB-003

8.3 Use Case Specification - UC-DB-005 (Brief - Backup)
MụcChi tiếtUC-IDUC-DB-005TênVận hành Backup & Restore Tự Động (SQL Server + PostgreSQL)Actor chínhSystem_Scheduler (Hangfire)Actor phụDBAdmin (cấu hình + verify)Pre-conditionDB đang hoạt động, backup directory đã cấu hình với đủ dung lượngSQL Server - Daily FullHangfire trigger 00:00: BACKUP DATABASE AMS_DB TO DISK='...\AMS_full_YYYYMMDD.bak' WITH COMPRESSION, STATS=10 Verify: RESTORE VERIFYONLY FROM DISK='...' Alert email nếu thất bạiSQL Server - Hourly DiffBACKUP DATABASE AMS_DB TO DISK='...\AMS_diff_YYYYMMDD_HH.bak' WITH DIFFERENTIAL, COMPRESSIONPostgreSQL - Daily FullpgAgent job 00:00: pg_dump -Fc -d ams_db -f /backup/ams_full_$(date +%Y%m%d).dump Verify: pg_restore --list file.dump > /dev/null (exit 0 = OK) Alert email nếu thất bạiPostgreSQL - Hourlypg_dump -Fd -j 4 -d ams_db -f /backup/ams_dir_YYYYMMDD_HH/ (parallel directory format)RetentionLocal: giữ 30 ngày. Archive mỗi tháng: giữ 10 năm (Luật KT Điều 41)Weekly Test RestoreDBAdmin chạy: restore vào DB sandbox ams_test, verify Trial Balance, xoá sandboxPost-conditionFile backup tồn tại, verify OK, email xác nhận gửi DBAdminFR liên quanFR-DB-006, NFR-DB-004
